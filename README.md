# BNBU SWDW 2026 Team Project

## 项目简介

这是一个基于网页的 BNBU 文创商店项目。  
用户可以浏览不同类型的文创商品，进行注册、登录，并在登录后购买商品。  
本项目使用 PHP 和 MySQL 实现用户账户管理和商品购买数量更新。

## 主要功能

- 用户注册
- 用户登录
- 用户退出登录
- 管理员登陆
- 显示当前登录状态
- 管理员查看购买总数
- 管理员查看各用户购买数量
- 商品展示
- 商品购买
- 用户查看已购买物品以及总价
- 购买后更新数据库中的商品数量
- 基本网页导航功能

## 使用技术
